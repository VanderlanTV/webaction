PROCEDIMENTOS INSTRUTIVOS PARA GERAR ARQUIVOS SINTEGRA E PRÉ-VALIDAÇÃO NO SISTEMA INFORVIA,  VALIDADOR SINTEGRA, TRANSMITIR ARQUIVO PARA O FISCO E IMPRIMIR COMPROVANTE DE RECEBIMENTO.

  1. O ARQUIVO PODE SER GERADO NO SISTEMA SVENDAS ou SGES, pela opção SERVIÇOS DISPONÍVEL EM AMBOS. O sistema SGES será o utilizado como exemplo. 

 

            1.1    Digite a senha de acesso, e faça duplo click no ICONE SINTEGRA, COMO MOSTRADO:

 

            1.2    A JANELA GERAR ARQUIVO SINTEGRA SERÁ ABERTA, conforme  visto. E gere o arquivo com os seguintes parâmetros:

                1.2.1    Informe a data de movimento e data de arquivo. Para da inicial, informe o primeiro dia do mês, e para data final o último dia do mês. Informe datas iguais para os campos movimento e  arquivo.

 

               1.2.2    Verifique  marcações para ativar transmissão de notas fiscais de entrada, de saída e cupons fiscais.ao lado direito das datas. Para gerar o arquivo, precisa que no mínimo uma das três opções esteja marcado.

 

   

                       

                  Dica: O campo  movimento data inicial e final é para casos de transmissão parcial.

                            Ao lado direito das datas, há marcações para ativar transmissão de notas fiscais de entrada, de saída e cupons fiscais.

                            Na fase inicial de gerar arquivo, pode ser desmarcado duas opções, de forma que o sistema gere o arquivo apenas da parte

                        ativada. Este procedimento acelera a geração de arquivo para validação. De maneira  que você possa trabalhar com validação e

                        correção de apenas um item de cada vez, fazendo gerações e validações, quantas vezes forem necessárias.

 

                            Após as correções em cada grupo de registro, ative as três opções e gere o arquivo para validação e transmissão.

 

               1.2.3    CLICK NO ICONE {GERAR Arquivo CONV.57/95} . CONFIRME COM {SIM} e CLICK {OK} CASO O SISTEMA O PEÇA ESTA CONFIRMAÇÃO.

 

                1.2.4    AGUARDE O PROCESSAMENTO. Pode ser necessário alguns minutos, dependendo do volume de informações a ser          processada.

               

 

 

     

               1.2.5    O SISTEMA IRÁ MOSTRAR A MENSAGEM OK ARQUIVO  gerado conforme visto:

 

 

            2. VALIDAÇÃO DO ARQUIVO SINTEGRA   

 

            2.1 Localize o validador SINTEGRA no WINDOWS , pelo iniciar programas ou pelo atalho na área de trabalho:

           

 

 

           

               VALIDADOR O ARQUIVO ;

                2.2    LOCALIZE O arquivo pelo CNPJ + IE + Mes+ .TXT, na unidade  C:\ , CONFORME AMOSTRA ::

 

                        Amostra de nome de arquivo para validação:    C:\CNPJ000000000000IE00000000000-10.TXT

 

                2.3 Em seguida clicar em {validar} em clicar em OK {validação concluída em tantos segundos}

               

 

4.  O ARQUIVO PODERÁ SER TRANSMITIDO PELO PROGRAMA TED APÓS A VALIDAÇÃO ESTAR SEM ERROS como na janela mostrada abaxo, com zero registros rejeitados e zero de advertências.

 

 

            4.1 Mas pode acontecer do validador apontar erros das mais diversas causas, como CNPJ não confere, CFOP não válido para a operação, Inscrição Estadual inválida, ect.

 

               4.1.1 Para visualizar as críticas de erros e advertências, Click em Críticas  e navegue por cada um dos erros e ou advertências mostrados:

 

                .No evento mostrado, o Inscrição estadual está inválida para a UF MG.

                4.1.2 As críticas podem ser impressas para facilitar o trabalho de correção, basta clicar em imprimir:

               

                Na formatação do arquivo SINTEGRA, há distinção para registro 50 de entrada ou de saída, é identificado apenas por uma letra [T] para NF de terceiros ou por uma letra [P] de emissão própria. Esta é a forma que você dispõe para identificar  se o registro é de CLIENTE pela letra [P], ou FORNECEDOR pela letra [T].

 

 

        4.1.3 Localizado a origem  do erro,  o tipo e  onde ocorreu, você precisará fazer as correções necessárias no sistema INFORVIA no módulo respectivo conforme o tutorial para editar incorreções de registro.

 

            Corrigidas as inconsistências, volte na validação conforme orientado neste tutorial ou no de edição de registro.

 

            acesse deste ponto o tutorial de edição de registros.
