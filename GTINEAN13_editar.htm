I.Orientação sobre Uso do Código GTIN EAN pela Sefaz.am.gov.br

 

1.      O que é o GTIN ?

.       

GTIN, acrônimo para Global Trade Item Number é um identificador para itens comerciais

desenvolvido e controlado pela GS1, antiga EAN/UCC. GTINs, anteriormente chamado

códigos EAN, são atribuidos para qualquer item (produto ou serviço) que pode ser

precificado, pedido ou faturado em qualquer ponto da cadeia de suprimentos. O GTIN é 

utilizado para recuperar informação pré-definida e abrange desde as matérias primas até

produtos acabados.

GTIN é um termo “guarda-chuva” para descrever toda a familia de identificação das

estruturas de dados GS1 para itens comerciais (produgos e serviços). GTINs podem ter o

tamanho de 8, 12, 13 ou 14 digitos e podem ser construidos utilizando qualquer uma das

quatro estruturas de numeração dependendo da aplicação.

O GTIN-8 é codificado no código de barras EAN-8. GTIN-12  é mais comumente utilizado no

código de barras UPC-A, o GTIN-13 é codificado no EAN-13 e o GTIN-14 no ITF-14.

 

2. O que é cEAN ?

Código de barras GTIN (antigo código EAN) do produto que está sendo faturado na NF-e. O

GTIN poderá ser GTIN-8 (antigo EAN-8), GTIN-12 (antigo UPC), GTIN-13 (antigo EAN),

GTIN-14 (antigo DUN-14

 

3. O que é o cEANTrib ?

FAQ – Obrigatoriedade do GTIN 

 

Código de barras GTIN (antigo código EAN) do produto tributável, ou seja,  a unidade que é

utilizada para calcular o ICMS de Substituição Tributária, como por exemplo a unidade de

venda no varejo. GTIN poderá ser GTIN-8 (antigo EAN-8), GTIN-12 (antigo UPC), GTIN-13

(antigo EAN), GTIN-14 (antigo DUN-14).

4. Qual a diferença entre cEAN e cEANTrib?

Quando o produto faturado for o mesmo que o produto tributável o código enviado no cEAN

e no cEANTrib será o mesmo, caso sejam diferentes o cEAN é o código de barras GTIN

(antigo código EAN) do produto que está sendo faturado na NF-e e o cEANTrib será o código

de barras GTIN  (antigo EAN) do produto tributável, ou seja, a unidade que é utilizada para

calcular o ICMS de Substituição Tributária.

Por exemplo: Compra de um palete com 12 caixas de produtos e cada caixa contém 9

latinhas, onde a venda (faturamento) foi realizado em caixas e a unidade tributável é a lata.

O cEAN será o código de barras da caixa com 9 latas e o cEANTrib o código da lata. 

 

 

 

 

 

8. Quais os benefícios do GTIN na NF-e ?

• Automação no Recebimento

• Melhoria no Controle de estoque 

• Conferencia do Pedido enviado com a NF-e recebida 

• Código único para controle de produtos

 

 

11. Não sou o fabricante do produto, preciso preencher os campos cEAN e cEANTrib?

Sim. Se o produto comercializado na NF- possuir código de barras com GTIN ele deve ser

destacado no documento, seja o documento gerado pelo fabricante, distribuidor, revenda, varejo,

etc.

II.Manual operacional para implementação do código GTIN EAN13 no Sistema Inforvia

 

O manual contempla duas partes; primeira a edição do código interno numérico para o padrão Internacional GTIN EAN13; segundo, editar o código interno para o GTIN EAN13 na emissão da NF-e, pela exigência do Ajuste SINIEF 16/2010, a partir de 1º de julho de 2011, em todo o território nacional

           

1.      Edição do código interno numérico para o padrão Internacional GTIN EAN13;

A edição foi projetada e implementada de maneira a facilitar a edição sem impactar o processo de vendas, emissão de NF-e, Recebimento etc. Assim, com segurança e por parte o códigos poderão ser editados,  pela técnica processamento em lote com atualização e reinicio de novo lote.

 

1.1   Para editar o código interno para o GTIN EAN13, acesse o cadastro de produtos pelos Svendas ou SGES e informe a senha e siga as instruções

 

1.2    Marque a Opção GTIN EAN

Selecione os produtos para modificação por descrição, código, referência, marca etc pelos critérios permitidos pelo sistema;

Informe os códigos GTIN EAN no campo respectivo

Informe os códigos disponíveis para alteração.

 

Atualize o Lote de códigos informado

Após digitação dos códigos click em

        Aguarde a finalização da atualização pela mensagem. Após a mensagem os códigos estarão substituídos na tabelas do Banco de dados do sistema, como Cadastro de produtos, estoque, vendas, Cupons fiscais, Notas Fiscais etc. O código antigo estará preservado no campo OLDCODREC para verificações de documentos anteriores a modificação e por exigência da legislação do SPED EFD atual.

 

1.3    Após a atualização reinicie os códigos GTIN.EAN  em

  Aguarde o final do processamento.

 

E essencial a guarda backups do sistema para eventual auditoria de banco de dados

 

2.      Edição do código pela Emissão da NF-e

A venda pode ser processada pelo código antigo, finalizada e liberada para emissão da NF-e. No ato da emissão os códigos poderão ser alterados para o GTIN EAN. A modificação do código na emissão da NF-e, altera em definitivo o código antigo para o código novo.

 

2.1 Para editar os códigos internos, para novos códigos GTIN EAN, click em editar Itens antes da emissão da NF-e

 

 

   A tela de edição será mostrada

 

            2.1.1    No campo GTINEAN informe o novo código GTIN EAN, informe o campo GTINEANTrib se Você vender produtos em Caixa ou embalagens que contenha vários produtos unitários onde estes  por sua vez possuam códigos GTIN EAN individualizados. Caso Você compre por caixa, mas faça a conversão das unidades e preços na entrada, Não será necessário informar o GTIN Tributável.

            Após informar o código tecle <ENTER> para a próxima linha

 

                        Faça a emissão da NF-.e. O sistema automaticamente converterá os códigos antigos para os novos GTIN EAN informados. O código estará modificado para a NF-e atual e disponível para as próximas inclusive.
